# AviUtl2-boxblur
AviUtl2用のボックスブラーシェーダーです。

縦横にしかかけることはできないものの、負荷は軽く、動きの表現からイラストなどのコンポジットまで様々なものに利用できます。
縦横を併用したぼかしを重ねがけすることでガウスぼかしを疑似的に再現できます。
バージョンの更新により動作が変わる可能性はあります。

## 必要環境
- AviUtl2

## インストール
`@ボックスブラー_横.anm2`をAviUtl2のscriptフォルダに入れる

## 使い方
フィルタエフェクトから「ボックスブラー_横」を追加

## パラメータ
- 半径: ブラーの強さ (0〜30)
- エッジピクセルを繰り返す: 端のピクセルを引き伸ばす

## 応用
- 重ねがけ　横、縦、横、縦の重ねがけを繰り返すことでガウスぼかしに近い効果が得られる　横だけなどでも重ねがけは効果がある
